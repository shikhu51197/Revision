## MERN Commerce Platform

This repository now includes a production-ready MERN e-commerce starter with:

- Express + MongoDB API featuring Passport auth, Multer uploads, Stripe payments, Nodemailer emails, and admin tooling.
- React 18 + Vite client with Tailwind CSS, Redux Toolkit, React Router v6, Stripe Elements checkout, and admin pages.
- Shared Zod schemas/types to keep validation in sync between client and server.

See `MERN_ECOMMERCE_PLAN.md` for the full architecture blueprint and `apps/api` / `apps/web` for the implementation.
